==================================================

Tang 1 - UI:

- ViewController
- Cell

Tang 2 - Logic:

- RootRouter
- SessionManager

==================================================

Vi project dang tach 2 loai model:

Loai 1 - Domain/API model:
- User
- Course
- Lesson
- Enrollment

Loai nay dung cho:
- Service
- Controller
- Codable
- API

Loai 2 - Core Data entity:
- StoredAccount
- StoredSession
- StoredEnrollment
- StoredProgress
- StoredDemoState

Loai nay dung cho:
- luu local
- Core Data

Day la cach tach de hoc de hon:
- model hien thi thi cu la Swift struct
- model luu DB thi la Core Data entity
