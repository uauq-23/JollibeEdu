JOLLIBEEEDU - PHAN CONG 4 NGUOI VA LOGIC CODE

1. Tong quan project

Ten project:
JollibeEdu - Online Course Management iOS App

Cong nghe:
- UIKit + Storyboard
- Swift
- Firebase Firestore
- Core Data cache local
- SessionManager, RootRouter, AppSettingsManager theo Singleton
- Kieu to chuc chinh la MVC

Muc tieu cua app:
- Cho khach xem khoa hoc
- Cho student dang ky, thanh toan, hoc bai, theo doi tien do
- Cho admin CRUD user, course, category, lesson va xem report

2. Logic code tong the de hoc

Luong chay tong the:
UI Storyboard -> ViewController -> Service -> DemoDataStore -> FirebaseSyncManager/CoreDataManager

Giai thich:
- Main.storyboard chua cac scene, navigation, tab bar, outlet va segue
- ViewController nhan su kien tu button, table, collection va cap nhat giao dien
- Service xu ly nghiep vu tung nhom chuc nang nhu auth, course, lesson, report
- DemoDataStore giu trang thai du lieu hien tai cua app
- FirebaseSyncManager day du lieu len Firestore va keo du lieu cloud xuong
- CoreDataManager dung de cache local de app van co du lieu khi mo lai
- SessionManager giu user dang dang nhap, role hien tai va dieu huong theo role
- RootRouter quyet dinh vao guest flow, student flow hay admin flow

Thu muc quan trong:
- Controllers/Public: guest home, login, register, course list, course detail, payment
- Controllers/Student: dashboard, my courses, lesson list, lesson detail, profile, progress
- Controllers/Admin: users, courses, categories, lessons, reports
- Services: nghiep vu cua tung module
- Managers: session, firebase, core data, image loader, app settings
- Helpers: theme, router, localization, alert, custom helper
- Models: user, course, lesson, category, enrollment, payment, report
- Views/Cells: reusable cell va component hien thi

3. Nguyen tac chia deu cho 4 nguoi

Nguyen tac:
- Moi nguoi phu trach 1 cum logic ro rang
- Moi nguoi co file ownership de de bao ve do an
- Moi nguoi co it nhat 1 nhom chuc nang co thao tac du lieu ro rang
- Moi nguoi deu phai hieu luong UI -> Controller -> Service -> Data

6. Cau tra loi ngan gon khi giang vien hoi

Hoi:
App nay theo pattern gi?
Tra loi:
App theo MVC. Storyboard va custom views la View. ViewController nhan su kien va dieu huong. Models va Services xu ly du lieu. SessionManager, RootRouter, FirebaseSyncManager la cac singleton ho tro app.

Hoi:
Du lieu di nhu the nao?
Tra loi:
Nguoi dung thao tac tren giao dien, ViewController goi Service, Service cap nhat DemoDataStore, sau do luu local bang Core Data cache va sync cloud bang Firebase Firestore.

Hoi:
4 nguoi chia nhau ra sao?
Tra loi:
Nhom chia theo 4 cum logic lon: Auth/User, Course Discovery/Category/Payment, Student Learning/Lesson/Profile, va Admin Course/Reports/Data Layer/Firebase.
