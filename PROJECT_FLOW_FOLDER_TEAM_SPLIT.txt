JOLLIBEEEDU - FUNCTION FLOW, FOLDER EXPLANATION, TEAM SPLIT

Project name:
- OnlineCourseManagement
- App display name: JolibeeEdu
- UIKit + Storyboard iOS app

==================================================
1. TONG QUAN CHUC NANG
==================================================

JolibeeEdu la app hoc online co 3 nhom luong chinh:

1. Guest flow
- Khach vao app co the xem trang chu, xem danh sach khoa hoc, xem chi tiet khoa hoc
- Chua dang nhap thi co the:
  - Browse courses
  - Xem chi tiet khoa hoc
  - Dang ky tai khoan
  - Dang nhap
  - Quen mat khau

2. Student flow
- Sau khi dang nhap voi role student:
  - Vao StudentTabBarController
  - Xem Home
  - Xem Courses
  - Xem Dashboard
  - Xem My Courses
  - Xem Profile
- Student co the:
  - Dang ky khoa hoc mien phi hoac thanh toan khoa hoc co phi
  - Hoc lesson
  - Danh dau hoan thanh bai hoc
  - Theo doi progress
  - Sua thong tin ca nhan
  - Doi mat khau
  - Dang xuat

3. Admin flow
- Sau khi dang nhap voi role admin:
  - Vao AdminRootTabBarController
  - Quan ly Users
  - Quan ly Courses
  - Quan ly Categories
  - Xem Reports
  - Tu Courses co the vao Admin Lessons de quan ly lesson theo khoa hoc
- Admin co the dang xuat tu cac man admin

3. DATA FLOW VA LUU TRU
==================================================

3.1. Session va routing
- SessionManager.swift
  - Luu token
  - Luu current user
  - Xac dinh role
  - Dang xuat

3.2. Core Data
- CoreDataManager.swift
  - Luu session dang nhap hien tai
  - Mirror tai khoan
  - Mirror enrollment
  - Mirror progress
  - Giup mo lai app van nho duoc hoc vien da hoc den dau

3.3. Service layer
- APIClient.swift
  - URLSession
  - GET / POST / PUT / PATCH / DELETE
  - Header Bearer token
- AuthService.swift
  - Login / Register / Forgot / Change password
- UserService.swift
  - User CRUD va profile
- CategoryService.swift
  - Category CRUD
- CourseService.swift
  - Course CRUD / list / detail
- LessonService.swift
  - Lesson CRUD / list lesson theo khoa hoc
- EnrollmentService.swift
  - Enroll / unenroll / my courses
- PaymentService.swift
  - Tao payment / confirm payment
- ProgressService.swift
  - Update progress / get progress
- ReportService.swift
  - Lay so lieu va charts

3.4. Demo fallback
- DemoDataStore.swift
  - Dung lam local backend de app chay demo khi chua co API that
  - Seed san user, course, lesson, enrollment, payment, progress
  - Dong bo du lieu voi Core Data
