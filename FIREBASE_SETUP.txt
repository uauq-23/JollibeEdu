Firebase setup for JolibeeEdu

1. In Xcode, add Swift Package:
   https://github.com/firebase/firebase-ios-sdk

2. Add these package products to the app target:
   - FirebaseCore
   - FirebaseFirestore

3. In Firebase Console:
   - Create an iOS app with bundle id: uauq.JollibeEdu
   - Download GoogleService-Info.plist

4. Drag GoogleService-Info.plist into:
   /Users/nguyenthuydung/Desktop/JollibeEdu/JollibeEdu
   and make sure target JollibeEdu is checked.

5. Firestore document used by the app:
   collection: app_state
   document: demo_state

6. App behavior after setup:
   - AppDelegate configures Firebase automatically.
   - SceneDelegate bootstraps remote state before routing.
   - DemoDataStore keeps local cache and syncs the latest state to Firestore.

7. If the app is already installed on simulator:
   - delete the app from simulator
   - run again after adding GoogleService-Info.plist
