JOLLIBEEEDU - PHAN CONG 4 NGUOI + LOGIC CODE
==================================================

1. MUC TIEU FILE NAY
==================================================

File nay dung de:
- Nho nhanh project dang duoc to chuc nhu the nao
- Hieu luong logic code chay tu dau den cuoi
- Chia viec cho 4 nguoi de lam cung project ma it dung file nhau
- Nho rang du lieu demo hien dang duoc persist bang Core Data


2. TONG QUAN KIEN TRUC
==================================================

Project dang di theo huong:
- UIKit + Storyboard cho scene/layout chinh
- Swift code cho logic
- URLSession + Service layer cho API
- DemoDataStore lam data source local fallback
- Core Data dung de luu state local

Thu muc chinh:
- Models
- Services
- Managers
- Helpers
- Views/Cells
- Controllers
- Base.lproj/Main.storyboard


3. LOGIC CODE CHINH
==================================================

3.1 App khoi dong

Luồng:
- App mo len
- RootRouter kiem tra SessionManager
- SessionManager doc session tu UserDefaults/Core Data
- Neu co token + user:
  - user.role == admin -> vao AdminRootTabBarController
  - nguoc lai -> vao StudentTabBarController
- Neu chua co session -> vao guest flow

File lien quan:
- JollibeEdu/Helpers/RootRouter.swift
- JollibeEdu/Managers/SessionManager.swift


3.2 Luong du lieu

Luồng goi du lieu:
- ViewController goi Service
- Service uu tien APIClient neu da config baseURL
- Neu chua co backend that -> fallback sang DemoDataStore
- DemoDataStore doc/ghi state local
- State local duoc persist bang Core Data

File lien quan:
- JollibeEdu/Services/APIClient.swift
- JollibeEdu/Services/*.swift
- JollibeEdu/Services/DemoDataStore.swift
- JollibeEdu/Managers/CoreDataManager.swift


3.3 Core Data dang luu gi

Hien tai Core Data dung de luu:
- Demo state tong:
  - account
  - category
  - course
  - lesson
  - enrollment
  - progress
  - payment
- Session hien tai
- Mirror account/enrollment/progress de de migrate va fallback

Entity logic:
- StoredDemoState
  - chua 1 blob state tong da encode
- StoredSession
  - token + user hien tai
- StoredAccount
  - user + password
- StoredEnrollment
  - userID + courseID + status + progress
- StoredProgress
  - lesson da hoc xong cua hoc vien

File lien quan:
- JollibeEdu/Managers/CoreDataManager.swift
- JollibeEdu/Services/DemoDataStore.swift


3.4 Logic dang nhap

Luồng:
- LoginViewController lay email/password
- AuthService login
- Neu dang demo mode:
  - AuthService -> DemoDataStore.login(...)
- SessionManager.saveSession(...)
- RootRouter.routeAfterAuthentication(...)

File lien quan:
- JollibeEdu/Controllers/Public/LoginViewController.swift
- JollibeEdu/Services/AuthService.swift
- JollibeEdu/Managers/SessionManager.swift


3.5 Logic guest -> course -> enroll

Luồng:
- GuestHomeViewController hien popular course
- CourseListViewController hien list + search + filter category
- CourseDetailViewController hien chi tiet
- Neu chua login -> an enroll se day qua Login
- Neu khoa hoc free -> enroll truc tiep
- Neu khoa hoc paid -> qua PaymentViewController

File lien quan:
- JollibeEdu/Controllers/Public/GuestHomeViewController.swift
- JollibeEdu/Controllers/Public/CourseListViewController.swift
- JollibeEdu/Controllers/Public/CourseDetailViewController.swift
- JollibeEdu/Controllers/Public/PaymentViewController.swift


3.6 Logic hoc bai

Luồng:
- Student vao My Courses
- Chon course -> LessonList
- Chon lesson hop le -> LessonDetail
- LessonDetail load video/anh
- Khi xong bai -> ProgressService.updateProgress(...)
- DemoDataStore cap nhat progress entry
- DemoDataStore update enrollment progress
- Man hinh My Courses / Dashboard reload lai de dua khoa hoc sang In Progress hoac Completed

File lien quan:
- JollibeEdu/Controllers/Student/MyCoursesViewController.swift
- JollibeEdu/Controllers/Student/LessonListViewController.swift
- JollibeEdu/Controllers/Student/LessonDetailViewController.swift
- JollibeEdu/Services/ProgressService.swift
- JollibeEdu/Services/DemoDataStore.swift


3.7 Logic media lesson

Lesson hien ho tro:
- link YouTube
- link Google Drive public
- link Dropbox public
- link direct mp4/mov/m3u8
- file local bundle://...
- anh bai hoc fallback neu video chua co

File lien quan:
- JollibeEdu/Helpers/MediaURLResolver.swift
- JollibeEdu/Managers/ImageLoader.swift
- JollibeEdu/Controllers/Admin/AdminLessonsViewController.swift
- JollibeEdu/Controllers/Student/LessonDetailViewController.swift


3.8 Logic admin CRUD

Admin co cac cum:
- Users
- Courses
- Categories
- Lessons
- Reports

Nguyen tac:
- Admin controller -> Service
- Service -> APIClient hoac DemoDataStore
- Sau khi CRUD xong -> reload list va cap nhat Core Data qua DemoDataStore.persist()

File lien quan:
- JollibeEdu/Controllers/Admin/*.swift
- JollibeEdu/Services/*.swift
- JollibeEdu/Services/DemoDataStore.swift
